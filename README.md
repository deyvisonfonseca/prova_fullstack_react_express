# 🚀 SisFullStack - Aplicação Completa Conteinerizada (P2)

Esta aplicação é o resultado do desenvolvimento integrado de uma API Backend (Node.js/Express) e de uma interface Frontend (React/Tailwind CSS), operando com persistência de dados poliglota (SQL e NoSQL) e orquestrada de ponta a ponta via Docker.

## 🛠️ Tecnologias e Arquitetura
- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Node.js, Express, JWT, OWASP Top 10 Security
- **Bancos de Dados:** MySQL (SQL Relacional) e MongoDB (NoSQL)
- **Infraestrutura:** Docker e Docker Compose

---

## 🐳 Como Executar a Aplicação Completa com Docker

O ecossistema foi totalmente automatizado. Não é necessária nenhuma instalação manual de dependências (`npm install`) ou execução local de serviços.

### 📋 Pré-requisitos
Certifique-se de ter o **Docker** e o **Docker Compose** instalados na sua máquina.
